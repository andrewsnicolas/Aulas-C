# 📚 Algoritmos em C — Exercícios da Faculdade

Bem-vindo ao meu repositório de exercícios da disciplina de Algoritmos utilizando a linguagem C. 😁
Aqui estão reunidos exercícios propostos pelos professores, desafios práticos e testes feitos durante as aulas da faculdade.

O objetivo deste repositório é acompanhar minha evolução na lógica de programação, resolução de problemas e desenvolvimento de algoritmos eficientes.

## 🧠 Conteúdos Estudados

Atualmente, os exercícios abordam temas como:

- [x] Entrada e saída de dados
- [x] Variáveis e tipos primitivos
- [x] Operadores matemáticos e lógicos
- [x] Estruturas condicionais (if, else, switch)
- [x] Estruturas de repetição (for, while, do while)
- [x] Vetores e matrizes
- [x] Funções
- [x] Lógica de programação

## 🛠️ Tecnologias Utilizadas
1. Linguagem C
2. GCC Compiler
3. CLion
4. Linux / Windows Terminal

## ▶️ Como Executar os Exercícios

Compile o arquivo:
```shell
gcc nome_do_arquivo.c -o programa
```
Execute:
```shell
./programa
```
No Windows:
```bash
programa.exe
```
